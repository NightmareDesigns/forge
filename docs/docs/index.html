Getting Started

Nightmare Designs Forge is a powerful design and cutting software that works with Cricut machines, plotters, and vinyl cutters.

System Requirements

  • Windows 10/11, macOS 10.14+, or Linux
  • 4GB RAM minimum (8GB recommended)
  • 500MB disk space
  • USB port for device connection

Installation

From Release

Download the latest release for your platform from the download page.

From Source

# Clone the repository
git clone https://github.com/nightmaredesigns/forge.git
cd forge

# Install dependencies
npm install

# Start the application
npm start

Build for Production

npm run build

Connecting Devices

Supported Devices

  • Cricut Machines - Explore, Maker, Joy (via USB)
  • Silhouette Cutters - Cameo, Portrait
  • Generic Plotters - Any HPGL-compatible device

Connection Steps

  1. Connect your device via USB
  2. Click Scan in the Device panel
  3. Select your device from the dropdown
  4. Click Connect

Image Tracing

Convert raster images (PNG, JPG, BMP) to vector paths ready for cutting.

Trace Options

Option Description
Threshold Controls black/white cutoff (0-255)
Detail High, Medium, or Low path detail
Posterize Multi-color trace with adjustable steps
Invert Swap black and white areas
